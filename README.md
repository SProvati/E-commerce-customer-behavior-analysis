# E‑commerce Customer Behavior Analysis
This repository contains a project for analyzing customer behavior in an e‑commerce setting. The analysis focuses on customer purchase patterns, device preferences, payment methods, and delivery performance to extract insights that help understand what drives customer satisfaction and engagement.

## 📂 Repository Contents
- `basic.ipynb` – Initial exploratory analysis of customer features and behavior.
- `data_preprocessing.ipynb` – Notebook showing how the dataset was cleaned and prepared for analysis.
- (Other notebooks/files as available)

## 🧠 Project Overview
The objective of this project is to gain insights into customer shopping behavior and experiences using real‑world transaction or user interaction data. This analysis may include:
- Data cleaning and preprocessing.
- Exploring how **device type** affects user interactions and purchases.
- Understanding **payment preferences** and their relationship to purchase behavior.
- Evaluating **delivery performance** across different locations and its impact on satisfaction.
- Visualizations to highlight key behavioral patterns.
